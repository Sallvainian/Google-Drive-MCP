Privacy Policy

Last updated: February 11, 2026

Overview

Personal Productivity Tools ("the App") is a personal-use application that integrates with Google Workspace APIs (Gmail, Google Drive, Google Docs, Google Sheets) for productivity automation.

Data Collection and Use

The App accesses Google Workspace data solely on behalf of the authenticated user. No data is collected, stored on external servers, shared with third parties, or used for any purpose other than the user's own productivity tasks.

Data Storage

OAuth tokens are stored locally on the user's own device. No user data is transmitted to or stored on any external server.

Third-Party Access

The App does not share, sell, or transfer any user data to third parties.

Google API Services

The App's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
